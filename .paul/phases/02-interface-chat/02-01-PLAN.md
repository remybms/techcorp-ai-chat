    - type Role = 'user' | 'assistant'
    - interface Message { id: number; role: Role; content: string; isLoading?: boolean }

    State : messages (Message[]) avec message accueil, input (string), isLoading (bool), isConnected (bool)
    Ref : messagesEndRef pour auto-scroll

    Fonctions :
    - scrollToBottom() via messagesEndRef.current?.scrollIntoView
    - handleSubmit(e: React.FormEvent) : preventDefault, guard input vide, ajouter message user, vider input, setIsLoading(true)
    useEffect : scrollToBottom sur changement messages

    JSX structure :
    div.app > header.header + main.messages + form.input-area
    - header : h1 "TechCorp AI — Phi-3.5-Financial" + div.status-dot (class connected si isConnected)
    - messages : map sur messages → div.message.{role} (+ .loading si isLoading)
    - form : input[text] + button[submit] disabled si isLoading ou input vide, texte "..." si isLoading sinon "Envoyer"

    Imports : useState, useRef, useEffect depuis react
    Pas de nouvelles dépendances
